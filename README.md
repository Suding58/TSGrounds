TSGrounds 🗺️

TSGrounds คือเครื่องมือสำหรับ ดูและแก้ไขโครงสร้างแผนที่ ของเกม TS Online และ TS2 Online 💎
ช่วยให้ผู้เล่น นักพัฒนา หรือ modder สามารถ สำรวจ แก้ไข และส่งออกข้อมูลแผนที่ ได้อย่างสะดวกและเข้าใจง่าย

คุณสมบัติหลัก ✨

ดูโครงสร้างแผนที่ (View Map Structure)

แสดงข้อมูลพื้นฐานของแผนที่ในเกม

แสดง Grid ของพื้นที่ (Grid Visualization)

แบ่งพื้นที่เดินได้ 🚶‍♂️, ห้ามเดิน ❌, น้ำ 💧

แก้ไขพื้นที่บนแผนที่ (Edit Map Areas)

เปลี่ยนประเภทพื้นที่หรือข้อจำกัดการเดินได้โดยตรง

ส่งออกเป็นรูปภาพของแผนที่ (Export Map Images)

บันทึกเป็น PNG / JPG / BMP

แก้ไขและส่งออกไฟล์ ground.mmg (กำลังพัฒนา) 💾

วิธีติดตั้งและใช้งาน 🛠️

ดาวน์โหลดไฟล์ TSGrounds.exe จาก repository นี้

วางไฟล์ .exe ไว้ใน โฟลเดอร์เกม TS Online หรือ TS2 Online ของคุณ

เปิดโปรแกรม TSGrounds.exe

ไปที่ File > Load เพื่อโหลดข้อมูลแผนที่

ใช้เครื่องมือเพื่อ ดู แก้ไข และส่งออกแผนที่

หมายเหตุ: ให้แน่ใจว่าไฟล์ ground.mmg และไฟล์รูปภาพอยู่ในโฟลเดอร์เดียวกับไฟล์ exe เพื่อให้โหลดได้ถูกต้อง

เริ่มต้นใช้งาน 🚀

เปิดแผนที่ผ่าน File > Load

Grid จะแสดง พื้นที่เดินได้, ห้ามเดิน, น้ำ อย่างชัดเจน

คลิกบนเซลล์เพื่อแก้ไขประเภทพื้นที่

ใช้ Export Image เพื่อบันทึกรูปภาพแผนที่

ฟีเจอร์ Export Ground.mmg สำหรับบันทึกการแก้ไขกลับไปยังไฟล์เกม (กำลังพัฒนา)
